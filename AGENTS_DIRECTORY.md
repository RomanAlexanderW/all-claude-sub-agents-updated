## Key Consolidations

### All Prediction Agents (44 total)
All prediction and forecasting agents are now consolidated in `05-predictions-forecasting/`:
- **Sports**: Baseball, basketball, football, tennis, player performance
- **Market/Financial**: Stock, crypto, economic, sales forecasts
- **Social/Political**: Elections, social movements, demographics
- **Technology**: AI development, climate, trends, tech adoption
- **Entertainment**: Viral content, music, movies, memes

### All Payment Integration Agents (17 total)
All payment processing agents are now consolidated in `03-business/finance/payments/`:
- **Major Platforms**: Stripe, PayPal, Braintree, Square
- **Mobile Payments**: Apple Pay, Google Pay
- **Regional**: WeChat Pay, Alipay, Venmo
- **BNPL**: Klarna, Afterpay
- **Other**: Crypto payments, ACH transfers, Authorize.Net

## Search Options

Use the `agents-search.py` tool to find agents:

```bash
# Search by category
python scripts/agents-search.py --category "backend"
python scripts/agents-search.py --category "predictions"
python scripts/agents-search.py --category "payments"

# Full-text search
python scripts/agents-search.py --query "payment stripe"
python scripts/agents-search.py --query "react testing"

# Show statistics
python scripts/agents-search.py --stats

# List all categories
python scripts/agents-search.py --list-categories

# Generate JSON index
python scripts/agents-search.py --generate-index
```
